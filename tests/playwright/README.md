# Playwright Actions

This folder contains action payloads for the shared web game Playwright client.

- `actions/smoke.json` - minimal start + short movement check
- `actions/fix-controls.json` - directional movement and collision-path scenario
- `actions/fix-idle.json` - idle scenario to validate enemy pursuit
- `actions/left-only.json` - repeated left movement regression check
- `actions/turn-up.json` - directional orientation check (up/down/left transforms)
- `actions/user-fixes.json` - broader regression for recent gameplay fixes
